# volume-create-cli-positional

`databricks volumes create` takes 4 positional args: CATALOG SCHEMA NAME VOLUME_TYPE. VOLUME_TYPE must be `MANAGED` or `EXTERNAL` — it is NOT a `--volume-type` flag. Confirmed against CLI v0.298.0.
